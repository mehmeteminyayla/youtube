# 00_setup
Genel gidişattan ve hazırlıklardan bahsediyoruz.

## Bölüm Özeti
Genel bir giriş ve hazırık bölümü.

## Açıklamalar
- `notebooks/`: Ders sırasında kullanılan notebook dosyaları
- `src/`: Tekrar kullanılabilir Python fonksiyonları/modülleri
- `data/raw/`: Ham örnek veri (küçük boyutlu)
- `data/processed/`: İşlenmiş veri çıktıları (küçük boyutlu)
- `assets/`: Grafikler, figürler ve görsel çıktılar

## Komutlar
```bash
# Örnek: sanal ortam
python -m venv .venv
source .venv/bin/activate

# Örnek: bağımlılık kurulumu
pip install -r requirements.txt

# Örnek: notebook başlatma
jupyter lab
```
